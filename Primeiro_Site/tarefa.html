Blockchain

A blockchain é uma base de dados distribuída que é mantida por uma rede de computadores. Os dados são armazenados em blocos, que são ligados uns aos outros usando criptografia. Cada bloco contém um hash do bloco anterior, o que torna impossível alterar os dados sem alterar todos os blocos subsequentes. As blockchains são usadas para registar transações de forma segura e transparente. Elas são também usadas para criar aplicações descentralizadas (dApps), que são aplicações que não são controladas por uma única entidade. As blockchains têm sido utilizadas para uma variedade de aplicações, incluindo a gestão de identidades, a rastreabilidade de produtos, e a criação de moedas digitais. A blockchain é uma tecnologia relativamente nova,mas tem o potencial de transformar a forma como fazemos negócios e interagimos com o mundo digital.
